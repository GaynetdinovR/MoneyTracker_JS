# Money Tracker
Money Tracker - Веб-приложение для отслеживания доходов

## Стек технологий

+ REACT
+ SASS
+ WEBPACK
+ REDUX

## Установка

+ Клонировать проект

```
git clone https://github.com/GaynetdinovR/MoneyTracker_JS.git
```

+ Установить необходимые пакеты

```
npm install
```

+ Запустить локальный сервер

```
npm start
```
